Identify the following ETF tickers and provide details for each.

Tickers to identify:
{tickers}

For each ticker, determine:
1. **region**: The primary geographic exposure. Must be one of: US, Europe, Japan, EM, Pacific, Gold
   - US: S&P 500, NASDAQ, US total market, or >50% US exposure
   - Europe: European stocks (Stoxx 600, MSCI Europe, individual European countries, **UK equities**)
   - Japan: Japanese stocks (Nikkei, TOPIX, MSCI Japan)
   - EM: Emerging Markets only (MSCI EM / EM IMI, individual EM countries). Note South Korea is Emerging under MSCI.
   - Pacific: **developed** Asia-Pacific ex-Japan — **MSCI Pacific ex Japan** (Australia, Hong Kong, Singapore, New Zealand). These are Developed markets per MSCI and FTSE, so they are NOT Emerging Markets.
   - Gold: Gold ETFs, gold miners, precious metals
   - For global/world ETFs, assign based on largest regional weight (usually US)
   - Note: tickers like SXR1 (iShares Core MSCI Pacific ex Japan, XETRA) → Pacific; SPYF (SPDR FTSE UK All Share, XETRA) → Europe

2. **name**: Full ETF name

3. **isin**: ISIN code if known (format: 2 letters + 10 alphanumeric)

4. **ter**: Total Expense Ratio as decimal (e.g., 0.0020 for 0.20%)

5. **is_accumulating**: true if accumulating (reinvests dividends), false if distributing, null if unknown

6. **note**: Optional note for special cases (e.g., "Global ETF - 60% US weighted")

Return ONLY valid JSON in this exact format:
{{

    {{
      "ticker": "EXAMPLE",
      "region": "US",
      "name": "Example ETF Name",
      "isin": "IE00XXXXXXXXXX",
      "ter": 0.0020,
      "is_accumulating": true,
      "note": null
    }}
  ]
}}

If you cannot identify a ticker, still include it with region "Unknown" and null for other fields.
